Import Mappings

Upload a JSON file containing mapping configurations. The file should contain an array of mapping objects.

Expected JSON Structure

[
  {
    "name": "Get Customer by ID",
    "priority": 5,
    "request": {
      "method": "GET",
      "path": "/api/customers/{id}",
      "headers": {
        "X-Tenant": "demo"
      }
    },
    "response": {
      "status": 200,
      "headers": {
        "Content-Type": "application/json"
      },
      "body": "{\"id\":\"123\",\"name\":\"Customer\"}",
      "templatingEnabled": true
    },
    "delays": {
      "mode": "fixed",
      "fixedMs": 100,
      "errorRatePercent": 0
    },
    "enabled": true,
    "tags": ["api", "customer"]
  }
]

Export Mappings

Download all current mappings as a JSON file for backup or sharing with other instances.

Export Current Mappings

This will download a JSON file containing all mappings in the current dataset.

Export Information

  • • All mappings from the active dataset will be included
  • • File will be in JSON format
  • • Includes all configuration: requests, responses, delays, etc.
  • • Can be imported into other API Simulator instances
  • • Perfect for backup and version control
